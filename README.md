para cuando el github lo clonen se van a la carpeta esa y abren el proyecto de la siguiente manera 
1. Abre una terminal dentro de esa carpeta y ejecuta:

npm install

Esto descargará React y todas las dependencias (tarda 1-2 minutos la primera vez).

3. Inicia la app:

npm run dev

4. Abre tu navegador en: http://localhost:3000
5. npm install react-router-dom
